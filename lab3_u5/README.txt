Вариант №У-5 “Научный журнал” (до 30 баллов):
 
Создайте иерархию классов, моделирующих информацию о статьях в научном журнале:
 
 класс Scientist (ученый):
полное имя;
ученая степень (без степени, кандидат наук, доктор наук);
 
 класс Article (статья):
название;
количество страниц, формул, ссылок;
набор ученых-соавторов;
набор ссылок на другие статьи;
 
 класс JournalRelease (выпуск журнала):
уникальный номер;
год и месяц выхода;
набор статей;
 
 класс Journal (журнал):
название
набор выпусков.
 
По правилам журнала, новая статья начинается с новой страницы, даже если последняя страница предыдущей статьи не заполнена до конца.
 
Напишите тестовую консольную программу, создающую и использующую данную модель:
создать достаточное количество тестовых данных в виде объектов модели - авторов, статей, выпусков, журналов.
распечатать количество страниц в каждом выпуске журнала;
распечатать список из 5 активных соавторов статей журнала в порядке убывания (фамилия соавтора - и количество статей, в которых он участвовал);
распечатать список из 5 наиболее математизированных статей, судя по количеству формул (вывести количество формул, статья, выпуск, журнал);
распечатать список статей, где все соавторы не обладают никакой ученой степенью;
распечатать список соавторов, на которых хотя бы раз сослались другие соавторы;
распечатать список статей, нарушающих правила ссылок и пояснения к нарушениям в виде произвольных сообщений об ошибках:
ссылка на статью в том же выпуске или в будущем выпуске;
ссылка статьи на саму себя;
корректно уничтожить все объекты модели.
